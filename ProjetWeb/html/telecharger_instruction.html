TELECHARGER

Telecharger le jeu,

Pour ceux qui utilisent Linux :

1) Entrez dans le Terminal et ecrivez cette commande : git clone https://github.com/PUFAp1Projets/HuongThang.git. Elle sert a telecharger tous les fichiers contenant dans le repertoire HuongThang. Le jeu Ping Pong se trouve (evidemment) dans le repertoire PingPong.

2) Apres avoir clone, entrez dans le repertoire HuongThang -> PingPong. Vous trouverez ici :

+ le repertoire Images : les images servant a l'affichage graphique (la balle, les gallets, l'image a fond...)

+ le fichier PingPongFinal.cpp : les codes en C++ et en SDL, le fichier principal de notre projet. Ecrivez cette commande pour faire marcher le jeu : g++ -o PingPongSDL PingPongFinal.cpp -lSDL

+ le fichier ping_pong_terminal.cpp : les codes en C++, le programme ne marche que dans Terminal, sans presentation graphique. Ecrivez cette commande pour faire marcher le jeu : g++ -o PingPongTer ping_pong_terminal.cpp

+ le fichier readme.txt : un texte presentant le projet complet.

Pour ceux qui utilisent Windows :

Veuillez entrez dans le repertoire HuongThang -> PingPong (cliquez sur le nom du repertoire afin d'aller vers le lien du GitHub) et copier tous les fichiers (y compris des images) a votre ordinateur.
