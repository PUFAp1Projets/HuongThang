II. PRESENTATION DU PROJET :
    
    1) Objectif :
    - Il nous faut programmer le jeu Ping Pong, ecrit en C++ avec la librairie SDL
    - Delai : 15 Decembre 2013


    2) Le contenu de notre projet :
    - Le fichier PingPongFinal.cpp est celui contenant des codes complets pour le jeu Ping Pong.
    - Le directory "Images" se compose des images utilisees en vue de la presentation graphique des codes dans le fichier PingPongFinal.cpp  
    - Le fichier ping_pong_terminal.cpp montre aussi les codes destines au jeu Ping Pong, mais celui-ci ne marche que dans le Terminal. (c'est comme un brouillon de notre groupe, pour mieux comprendre le jeu avant la programmation en SDL)


    3) La regle du jeu :
    - Le theme : Noel
    - Dans le cadre, se deplace une balle dont la vitesse augmente automatiquement toutes les 15 secondes. Les joueurs doivent controler les gallets de maniere que la balle ne touche pas les murs verticales. Si cela se produit, un des jouers recoit 1 point. Apres 3 tours, le jeu termine et le resultat (qui gagne, qui perd) s'affiche sur l'ecran.
